Manual Wizard Test

Instructions:

  1. Open Developer Tools (F12)
  2. Navigate to the Console tab
  3. Run the following commands:
// Step 1: Set auth tokens
localStorage.setItem('family_hub_access_token', 'manual-test-token');
localStorage.setItem('family_hub_token_expires', new Date(Date.now() + 3600000).toISOString());

// Step 2: Navigate to wizard
window.location.href = 'http://localhost:4200/family/create';

// Step 3: After page loads, verify wizard (run in console)
const diagnostic = {
    hasWizard: !!document.querySelector('app-wizard'),
    hasTitle: document.body.textContent.includes('Create Your Family'),
    hasInput: !!document.querySelector('input[aria-label="Family name"]'),
    hasProgressBar: !!document.querySelector('app-progress-bar'),
    wizardHTML: document.querySelector('app-wizard')?.innerHTML.length
};
console.table(diagnostic);

Expected Results:

  • ✅ hasWizard: true
  • ✅ hasTitle: true
  • ✅ hasInput: true
  • ✅ hasProgressBar: true
  • ✅ wizardHTML: > 1000

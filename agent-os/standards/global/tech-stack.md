# Tech Stack

Family Hub uses the following technology stack.

## Frontend

- Angular 21 with TypeScript
- Tailwind CSS for styling
- Apollo Client for GraphQL

## Backend

- .NET Core 10 with C# 14
- Hot Chocolate GraphQL 14.1.0
- MediatR for CQRS
- Vogen 8.0+ for value objects

## Database

- PostgreSQL 16 with Row-Level Security (RLS)
- EF Core 10 for ORM

## Infrastructure

- Docker Compose (local development)
- Kubernetes (production)
- RabbitMQ for messaging
- Redis for subscriptions
- Zitadel for OAuth 2.0

# Tech Stack

## Backend

- .NET 9 with C#
- Hot Chocolate GraphQL
- Wolverine for CQRS and messaging (NOT MediatR)
- Vogen for value objects
- FluentValidation

## Frontend

- Angular 19 with TypeScript
- Tailwind CSS
- Apollo Client for GraphQL

## Database

- PostgreSQL with Row-Level Security (RLS)
- EF Core for ORM

## Auth

- Keycloak for OAuth 2.0 / OIDC (NOT Zitadel)

## Infrastructure

- Docker Compose (local dev)
- Kubernetes (production target)
